Practico Segundo parcial

Ejercicio 1.- Realizar el diagrama de flujo para mostrar todos los números impares de un numero ingresado por el usuario

Ejercicio 2.- Desarrollar un diagrama de flujo donde se ingresen dos números y te muestre cual es el número mayor

Ejercicio 3.- Desarrolle un diagrama de flujo que realice la sumatoria de los números enteros comprendidos entre el 1 y el 10

Ejercicio 4.- Realizar un diagrama de flujo para sumar todos los números pares entre 7 y 15

Ejercicio 5.- Desarrolle un diagrama de flujo que permita determinar el área y volumen de un cilindro dado su radio (R) y altura (H)

Ejercicio 6.- Desarrollar un diagrama de flujo donde se ingresen tres números y te muestre cual es el número mayor

Ejercicio 7.- Desarrolle un diagrama de flujo que permita leer un número y muestre si el número es positivo o negativo, en caso del número 0, mostrar que es un número “neutro”

Ejercicio 8.- Escribir los números Pares del 1 al 15 utilizando For

Ejercicio 9.- Desarrollar un diagrama de flujo donde se ingresen dos números y te muestre cual es el número menor
